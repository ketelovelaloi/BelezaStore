# BelezaStore

## Project Documentation

Welcome to the BelezaStore project!

### Overview
BelezaStore is an innovative online platform that allows users to buy and sell beauty products.

### Features
- User-friendly interface
- Secure payment processing
- Comprehensive product catalog
- User reviews and ratings
- Order tracking

### Installation
To install BelezaStore, clone the repository and run the following command:
```
npm install
```

### Usage
To start the application, use the command:
```
npm start
```
